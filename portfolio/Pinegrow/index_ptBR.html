Recursos

Você desenvolve páginas em HTML, temas para WordPress, Bootstrap ou Foundation?
Veja porque o Pinegrow deve fazer parte de suas ferramentas:

Abra arquivos HTML no seu computador

O Pinegrow permite trabalhar em arquivos HTML direto no seu computador. Ele não adiciona nenhum framework, leiaute ou estilo próprio no seu código. E não precisa subir, importar ou exportar nada. Apenas abra, edite e salve seus arquivos HTML.

Acelere seu desenvolvimento HTML

Crie rapidamente o leiaute de seu site web com as poderosas ferramentas visuais do Pinegrow para adicionar, editar, mover, clonar e apagar elementos HTML.

Loading animated gif... Open basic page instead.

Edição em tempo real

Tudo no Pinegrow é feito em tempo real. Não existem "previews". Você pode editar e testar o resultado ao mesmo tempo - mesmo se você estiver utilizando elementos JavaScript dinâmicos.

Loading animated gif... Open basic page instead.

Bootstrap, Foundation, AngularJS ou HTML

Pinegrow tem suporte completo para Bootstrap e Foundation. Arraste componetes na página e customize as propriedades na IU. O Pinegrow automaticamente seleciona os atributos e adiciona classes, e até ajusta o código HTML. AngularJS, 960 Grid e HTML puro também são suportados.

Loading animated gif... Open basic page instead.

Construtor de temas para WordPress

Abra ou crie uma página em HTML no Pinegrow Web Editor. Adicione ações WordPress nos elementos HTML e selecione os parâmetros. Exporte o tema para WordPress. O Pinegrow gera código PHP e divide a página em arquivos de tema em PHP. E ainda tem mais!

Loading animated gif... Open basic page instead.

Edição multi-páginas

Editar múltiplas páginas ao mesmo tempo deixa a criação de sites responsivos uma moleza. Duplique e espelhe páginas (com as mudanças da página original copiadas automaticamente na espelhada), escolha níveis de "zoom" ou tamanho de tela individualmente.

Loading animated gif... Open basic page instead.

O Pinegrow ama o código

O Pinegrow não esconde o código de você. Edite páginas visualmente e por código - ao mesmo tempo. Mudanças feitas na IU são refletidas no código. A edição de código - mesmo feita num IDE externo - são atualizadas imediatamente na página e na IU. (Aliás, isso torna o Pinegrow na ferramenta perfeita para aprender HTML & CSS).

Loading animated gif... Open basic page instead.

Editor CSS poderoso

Edite regras CSS visualemnte ou por código. Adicione variáveis e expressões para temas ficarem fáceis de personalizar. Use o Gerenciador de folhas de estilos para clonar, anexar ou remover estilos. Mudanças feitas nos estilos são vistas na hora em todas as páginas em que são utilizadas.

Loading animated gif... Open basic page instead.

Deixe suas páginas responsivas

Crie sites web responsivos facilmente com a ferramenta Media query. Adicione pontos de quebra personalizados ou deixe o Pinegrow criá-los analisando as folhas de estilo. Use a edição múlti-páginas para ajustar o site para telas de todos os tamanhos.

Loading animated gif... Open basic page instead.

Contrua sites web em instantes com Blocos

Use a coleção de blocos para sites prontos para usar para rapidamente criar lindos sites. Arraste blocos para a página, customize o conteúdo, mude os estilos se quiser - e o seu site está pronto! Você pode até criar um tema totalmente personalizável para WordPress usando blocos.

Perguntas Frequentes

O que o Pinegrow tem de diferente do Dreamweaver, Macaw, Jetstrap ou Webflow?

O Pinegrow é um app de desktop que trabalha com arquivos HTML. Nós acreditamos que um app pra computador funciona bem melhor no estilo de trabalho de desenvolvimento de sites do que os apps baseados na nuvem.

O Pinegrow tem recusrsos exclusivos como editar várias páginas ao mesmo tempo, a gama completa de componentes Bootstrap e Foundation, estilos CSS com suporte a LESS, gerenciador de folha de estilos ... e mais recursos estão para vir.

O Pinegrow não adiciona nenhum código próprio no HTML e CSS em que você está usando. Ele te ajuda a trabalhar com o HTML e CSS de maneira mais efetiva, seja usando a interface ou através de código.

Quais os termos da licença?

Uma licença única é necessária para cada usuário do app. Uma exceção é feita para membros de uma mesma família - não faz sentido precisar comprar uma licença só para suas crianças se elas quiserem brincar com o Pinegrow no seu computador.

Você pode instalar uma cópia do Pinegrow em até 3 computadores (em casa e no trabalho, no Windows, Mac ou Linux) conquanto se você estiver usando app. 

Se você usar o Pinegrow em sua empresa, precisa comprar uma licença para empresas. Você precisará de uma licença para casa usuário de Pinegrow na sua empresa.

Como é a atualização grátis durante 1 ano?

Cada versão comprada possui uma licença para usar o Pinegrow atual e inclui um ano de atualizações grátis das novas versões.

Depois de um ano as atualizações grátis expiram e caso você queira continuar atualizado, é necessário pagar - nesse caso a metade do preço uma licença - para ter mais 1 ano de atualizações.

A sua versão vai continuar funcionando sempre mesmo depois desse período, mas estamos confiantes que de você vai querer aproveitar todo o potencial das novas versões e o preço reduzido na renovação.

Por que o nome "Pinegrow"?

Nós amamos árvores, e amamos pinheiros. Eles tem um cheiro incrível e é muito legal ficarmos na sombra debaixo delas no verão. As folhas do pinheiro deixam a quantidade perfeita de luz do sol passar. E árvores crescem bastante, então "Pinegrow"!
